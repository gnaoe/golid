## ⚡ What is golid?

golid is a ready-to-use software that helps build websites quickly. It combines the Go programming language with SolidJS, a tool for creating web pages. golid includes features like user login, more than 70 premade web parts, fast page loading, live updates, and easy deployment with one command.

You do not need to know programming to try golid. This guide shows you how to get it up and running on your Windows computer step by step.

---

## 🖥️ System Requirements

Before downloading and running golid, make sure your computer meets these requirements:

- Windows 10 or newer
- 4 GB of RAM or more
- At least 500 MB free disk space
- Internet connection for downloading and updates

golid runs as a web application framework, so you will use a web browser (like Chrome, Edge, or Firefox) to access the applications you build.

---

## 🔍 Main Features

- **Built-in user login system.** Secure sign-in for web applications.
- **Over 70 components.** Pre-built buttons, forms, menus, and other webpage elements.
- **Server-side rendering (SSR).** Web pages load faster and better for search engines.
- **Real-time updates.** Changes on the web app happen instantly without reloading.
- **Easy deployment.** Upload your website with one command when ready.
- **Tech stack:** Go, SolidJS, PostgreSQL database, Tailwind CSS for styling, and TypeScript.
